
## How To Deploy On Google Cloud


## STEP 1 : ADD ENV

```
nano plugins/vars.py
```


USE : `ctrl + s` : FOR SAVE | `ctrl + x` : FOR EXIT

## STEP 2 : BUILD DOCKERFILE

```
docker build -t txt-leech .
```

## STEP 3 : RUN DOCKER APP
```
docker run -it --rm txt-leech
```
